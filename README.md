🪟 ИИ-АССИСТЕНТ ДЛЯ КОМПАНИИ ПО УСТАНОВКЕ ОКОН
Проект: «Окна»
📝 ОПИСАНИЕ ПРОЕКТА
Название: ИИ-ассистент для компании по установке окон «Окна»
Тип проекта: Виртуальный консультант на базе искусственного интеллекта
Платформа: Suvvy.ai
Язык: Русский
Целевая аудитория: Частные клиенты (квартиры, дома, дачи), коммерческие клиенты (офисы, магазины, бизнес-центры), застройщики и подрядчики
Основная задача: Автоматизация первичной коммуникации с клиентами, квалифицирование лидов и запись на бесплатный замер.
🎯 ЦЕЛИ И ЗАДАЧИ
Стратегические цели:
Увеличение конверсии в заявку +30-40%
Сокращение времени ответа клиенту < 5 секунд
Снижение нагрузки на менеджеров 40-50% звонков обрабатывает бот
Круглосуточная доступность 24/7 без перерывов
Операционные задачи:
Осмысленное ведение диалога — понимание контекста, удержание нити разговора
Ответы на вопросы по базе знаний — цены, сроки, гарантия, материалы, услуги
Запись клиента на встречу — сбор контактов, выбор удобного времени
Добавление встречи в Google Календарь — автоматическая синхронизация
Квалификация лида — определение типа клиента, бюджета, срочности
Передача сложных запросов менеджеру — эскалация при необходимости
⚙️ ФУНКЦИОНАЛ и ОСНОВНЫЕ ВОЗМОЖНОСТИ     
Сценарий использования - 
├─────────────────────────────────────────────────────────────┤
│  ✅ Приветствие и идентификация клиента: клиент пишет на сайт/в мессенджер → бот приветствует и выявляет потребность
│  ✅ Консультация по продукции (окна, двери, остекление) и расчёт ориентировочной стоимости: клиент задаёт вопросы → бот   отвечает из базы знаний, в случае сложного вопроса → бот переводит диалог на менеджера                          │
│  ✅ Запись на бесплатный замер и сбор контактных данных (имя, телефон, адрес): клиент готов → бот собирает контакты и назначает время, встреча автоматически добавляется в Google Календарь менеджера, за 24 часа до замера — автоматическое напоминание клиенту       
🛠 ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ
Qwen: создание системного промпта, генерация базы знаний, ответы на сложные вопросы (https://qwen.ai/)

✅ Создать единую базу знаний для всех каналов связи
